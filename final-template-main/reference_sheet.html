More HTML tags!

A div tag defines a section of an HTML page. For example, if I wanted to syle a particular section differently, I could put it inside a div and give that div some styles:

This section should have a red border.

An iframe, or inline frame, embeds another document or piece of code into your current HTML page. This is how we insert our Datawrapper charts into an HTML web page!

An img tag creates a space in your HTML page for an image. If you are adding an image into your page, your image must also exist in the same folder as the file where it's referenced, and you must define its location within the src attribute of your img tag.

List tags are also useful! You can have an ordered list, like this one:

  1. Freshman
  2. Sophomore
  3. Junior
  4. Senior

You can also create an unordered list, like this one:

  • Newhouse
  • Maxwell
  • Whitman
  • Falk

There are literally hundreds of HTML tags. You can learn more about each of them on W3Schools, a very commonly-used html reference page.
